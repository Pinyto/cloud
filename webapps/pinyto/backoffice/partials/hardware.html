There are plans to create a pre-installed mini server with Pinyto for your home. It would be a small machine with low power consumption capable of hosting Pinyto, a private website, Seafile and maybe your own e-mail server.

Since our reference design is not finished yet you have to be patient. If you want to use Pinyto and you do not want to wait then the best idea is to rent a server somewhere and install Pinyto there. Read how this is done here.
